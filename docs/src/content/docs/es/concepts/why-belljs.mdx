**Cansado de las alertas que ocupan todo el espacio en pantalla y rompen con el flujo de trabajo de tus usuarios?**, Pues para eso mismo creamos **BellJs** no una sino la libreria de alertas ligera y agnostica al entorno **js** que estes usando. **BellJs** incluye todo lo necesario para una libreria de alertas funcional, desde tipos de alertas hasta promesas y temas disponibles para personalizar al maximo esta libreria y ajustarla asi a tus necesidades, todo esto haciendo uso de una **API** intuitiva.

Algunas de los aspectos a destacar incluyen:
