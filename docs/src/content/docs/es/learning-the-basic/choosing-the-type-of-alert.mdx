El segundo parametro de **Bell** es un `String` y se refiere al tipo de alerta, dependiendo de cual elijas la alerta se tornara diferente, algunos de los tipos son:

- **`info`**: Alertas para mostrar informacion.
- **`error`**: Alertas para informar de algun error. 
- **`success`**: Alertas para mostrar que se logro un objetivo. 
- **`warning`**: Alertas para mostrar que sucedio algo, no necesariamente un error critico.
- **`promise`**: Alertas para indicar que algo esta cargando. 

Puedes usar cualquiera de estas en tus alertas aqui un ejemplo:

```js
new Bell({
	title: "Ocurrio un error grave",
},"error").launch()
```
